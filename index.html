Projects

DiveBook

A virtual logbook for SCUBA divers and dive shops where divers can enter logs of their dives and view analytical data on graphs and an interactive map for each of their dives. Dive shops can verify that recreational divers have completed the dives entered in their logbooks. The app focuses on data visualizations using D3, React BriteCharts, and React - MapGL, a QR code scanner implementation for verifying logs, and a Postgis spatial database for storing latitude and longitude points.

A full stack, JavaScript e-commerce site for musical instruments that can handle both logged in and guest users, manage their carts, and allow them to check out using the Stripe API and receive email confirmation of the order.
